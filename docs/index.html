Equity

1,022 tickers (51 US + 949 LSE ETFs + 13 macro ETFs + 9 factor ETFs). Daily OHLCV from 1980. Includes Vanguard, iShares, SPDR UCITS ETFs. 44 MB Parquet.

Crypto

14 tokens (BTC, ETH, SOL, BNB, stablecoins, Solana DeFi). Daily from 2014. 1.1 MB.

Macro

19 FRED series: SP500, VIX, Treasury yields, credit spreads, GDP, CPI, oil, USD. From 1947.

Factors

Fama-French FF3, FF5, Momentum. Daily + monthly. Back to 1926. Ken French Data Library.

Usage Examples

Interactive dashboard with code examples for all 6 datasets: equity, crypto, macro, factors, LSE ETFs, and metadata queries.

Macro Defense Rotation

Backtesting vignette: rotating among TLT/GLD/DBC/UUP to hedge macro risk. Expanding window methodology, in-sample vs out-of-sample holdout.

Factor MAX Rotation

Rotating among Fama-French factors using the MAX signal — maximum daily return predicts next-month factor returns. Based on Alpha Architect research (Dec 2025).

DRIF: Daily Return Information

Elastic net on the full distribution of past 21 daily factor returns (chronological + rank) predicts next-month returns. Subsumes MAX and momentum signals.

Stock-Level Backtests

MAX and DRIF signals applied to 660 individual stocks (S&P 500 + STOXX 600). Decile long-short portfolios. 4-strategy comparison: stock vs factor, MAX vs DRIF.

# Install from GitHub
pak::pak("JohnGavin/historical", subdir = "packages/historicaldata")

# Then:
library(historicaldata)
hd_ohlcv("AAPL", from = "2024-01-01")
hd_ohlcv(hd_group("FAANG"), from = "2024-01-01")  # batch query
hd_macro(c("SP500", "VIXCLS"), from = "2024-01-01")
hd_search("vanguard")  # search metadata

No R needed — query the same data directly from Python using DuckDB’s native hf:// protocol:

import duckdb

con = duckdb.connect()
df = con.sql("""
    SELECT date, close, volume
    FROM 'hf://datasets/dsfefvx/finance-historical-data/equity_daily.parquet'
    WHERE ticker = 'AAPL' AND date >= '2024-01-01'
    ORDER BY date
""").df()
# Enter reproducible Nix shell (all R + Python packages included)
nix develop github:JohnGavin/historical

Then start R and load the package. All dependencies (dplyr, ggplot2, scales, DT, arrow, duckdb) are pre-installed in the Nix shell.

# Clone, enter Nix shell, fetch data, run pipeline
git clone https://github.com/JohnGavin/historical.git && cd historical
nix develop
python scripts/fetch_equity.py     # 51 equity tickers
Rscript scripts/fetch_crypto.R     # 14 crypto tokens
Rscript scripts/fetch_macro.R      # 19 FRED series
Rscript scripts/fetch_factors.R    # Fama-French factors
t run src/pipeline.t               # T pipeline
