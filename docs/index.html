historicaldata

R package DuckDB + Parquet Hugging Face

Historical finance database for backtesting: equities, crypto, macro, and factors. Code-only R package — data hosted on Hugging Face, queried via DuckDB hf:// protocol with predicate pushdown.

Installation

# R (pak)
pak::pak("JohnGavin/historical", subdir = "packages/historicaldata")

# Then:
library(historicaldata)
hd_ohlcv("AAPL", from = "2024-01-01")
hd_ohlcv(hd_group("FAANG"), from = "2024-01-01")  # batch query
hd_macro(c("SP500", "VIXCLS"), from = "2024-01-01")
hd_search("vanguard")  # search metadata

# Python (DuckDB, no R needed)
import duckdb
df = duckdb.sql("""
  SELECT * FROM 'hf://datasets/dsfefvx/finance-historical-data/equity_daily.parquet'
  WHERE ticker = 'AAPL' AND date >= '2024-01-01'
""").df()

Datasets

Equity

1,013 tickers (51 US + 949 LSE ETFs + 13 macro ETFs). Daily OHLCV from 1980. Includes Vanguard, iShares, SPDR UCITS ETFs. 44 MB Parquet.

Crypto

14 tokens (BTC, ETH, SOL, BNB, stablecoins, Solana DeFi). Daily from 2014. 1.1 MB.

Macro

19 FRED series: SP500, VIX, Treasury yields, credit spreads, GDP, CPI, oil, USD. From 1947.

Factors

Fama-French FF3, FF5, Momentum. Daily + monthly. Back to 1926. Ken French Data Library.
